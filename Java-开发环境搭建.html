Java 开发环境搭建

发表于 2018-10-14 | 分类于 Java | | 阅读次数：
本文字数： 818 | 阅读时长 ≈ 1 分钟

前言

      本篇博客介绍的是在 Mac 上搭建Java开发环境。其实无论是在什么操作系统下，搭建Java开发环境无非都是安装JDK，然后再修改环境变量

下载与安装 JDK

  • 官网下载：JDK

配制环境变量

  1. 在终端执行：/usr/libexec/java_home -v，查看JDK的安装位置

  2. 编辑配置文件，写入 Java 环境变量的配置（zsh的可编辑.zshrc，bash可编辑.bash_profile）

  3. 终端输入：source ~/.bash_profile 应用变更

  4. 测试 Java 环境是否搭建成功

    终端输入：java -version，查看所安装的JDK的版本
